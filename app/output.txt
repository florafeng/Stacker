<p>Sounds like a pretty straightforward loop/recursion.</p>

<p><a href="http://jsbin.com/uLaZUBa/1/edit?js,console" rel="nofollow">Here you go</a></p>

<pre><code>function divideEvenly(numerator, minPartSize) {
  if(numerator / minPartSize&lt; 2) {
    return [numerator];
  }
  return [minPartSize].concat(divideEvenly(numerator-minPartSize, minPartSize));
}

console.log(divideEvenly(1000, 333));
</code></pre>

<p>To get it to be two decimals of currency multiply both numbers by 100 before calling this function then divide each result by 100 and call <code>toFixed(2)</code>.</p>

<p><a href="http://jsbin.com/uLaZUBa/2/edit?js,console" rel="nofollow">Like so</a></p>

<pre><code>function divideCurrencyEvenly(numerator, divisor) {
  var minPartSize = +(numerator / divisor).toFixed(2)
  return divideEvenly(numerator*100, minPartSize*100).map(function(v) {
    return (v/100).toFixed(2);
  });
}


console.log(divideCurrencyEvenly(3856, 3));
//=&gt;["1285.33", "1285.33", "1285.34"]
</code></pre>
